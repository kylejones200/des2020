Multiple Linear Regression

Multiple Linear Regression

The point of regression analysis is to use the relationship between a random variable $Y$ and another variable $x$ to make better predictions about $Y$ than could be made by simply using the overall average $\overline{y}.$ In the “real world,” variables are related to one another, and thus a model that allows the distribution of one variable to change with another is one way to model this relationship. However, it would be rather silly to conclude that $Y$ is influenced only by a single variable $x.$ Rather, it is much more sensible that $Y$ is influenced by a collection of $x$ variables, say $\mathbf{x=(}x_{1},x_{2},\ldots ,x_{p})$ where $p>2.$

For example, if $Y$ is a person’s cholesterol level, a person’s age would be a logical choice for an $x$ variable. We might call it $x_{1}.$ But cholesterol might also be related to gender $(x_{2})$, weight $(x_{3})$, blood pressure $(x_{4})$, and many other variables. Being able to model $Y$ as a function of all of these variables simultaneously would allow doctors and researchers to examine the effects of each of these variables on cholesterol level in the presence of the other variables, and make predictions that could help patients make decisions about their lifestyles.

Fortunately, regression analysis extends naturally from one independent variable to many independent variables. The basic concept is essentially the same, except we add more $x$ variables to the mix:

  • We assume the probability distribution of $Y$ changes as the collection of $x$ variables $\mathbf{x=(}x_{1},x_{2},\ldots ,x_{p})$ changes

  • Specifically, we assume that the mean (or expected value) of  $Y$, denoted as $E(Y)$, changes as $\mathbf{x}$ changes. To indicate this we write $E(Y|\mathbf{x})$ read as “the expected value of $Y$ given $\mathbf{x}$.”

When multiple $x$ variables are involved, however, we can define many types of functional relationships between $Y$ and $\mathbf{x.}$ The simplest one is known as the “first-order multiple linear regression model,” which we now define:

A first-order multiple linear regression model is specified by

$$Y=\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}+\varepsilon$$

where the $\beta$ terms are fixed (but unknown) parameters and $\varepsilon$ has a normal distribution with mean $0$ and standard deviation $\sigma$.

The implication of the multiple linear regression model is that the mean of $% Y$ is $E(Y|\mathbf{x})=\widehat{Y}=$ $\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}$ because, by the rules of expected value (see Handout $1$),

$$\begin{aligned} E(Y|\mathbf{x}) &=&E(\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}+\varepsilon ) \ &=&\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}+E(\varepsilon )\text{ [since }\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}\text{ is a constant]} \ &=&\beta_{0}+\beta_{1}x_{1}+\beta_{2}x_{2}+\ldots +\beta_{p}x_{p}\text{ [since we assume }E(\varepsilon )=0]\end{aligned}$$

We can interpret the parameters in a similar manner as with the simple linear regression model, except that we must now be mindful of the other $x$ variables in the model. Here are the general interpretations.

  • $\beta_{0}$ is the mean of $Y$ when all the $x$ variables are $0$.

  • $\beta_{1}$ is the increase in the mean of $Y$ for a one-unit increase in $x_{1}$, holding the other $x$ variables fixed$.$

  • In general, $\beta_{j}$ is the increase in the mean of $Y$ for a one-unit increase in $x_{j}$, holding the other $x$ variables fixed.

The parameters are estimated with ordinary least squares (OLS) just as with the simple linear regression model. However, in the case of multiple $x$ variables, the formulas are not simple to write out by hand. In fact, the values that minimize $SSE=\sum (y_{i}-b_{0}-b_{1}x_{i1}-b_{2}x_{i2}-\ldots -b_{p}x_{ip})^{2}$ are found using matrix algebra, which is beyond the scope of this course. Suffice it to say that you will never have to find these values by hand. Multiple regression is nearly always carried out using specialized software such as Excel.

With the simple linear regression model, the fitted function $\widehat{y}% =b_{0}+b_{1}x_{1}$ was a line. With a multiple linear regression model, the fitted function $\widehat{y}=b_{0}+b_{1}x_{1}+b_{2}x_{2}+\ldots +b_{p}x_{p}$ is a **plane** (flat surface) in $p+1$ dimensions. For instance if we were to fit a model with two $x$ variables, we would obtain the following function in Figure [regplane]{reference-type="ref” reference="regplane”}, a plane in three dimensions. With more than two $x$ variables, we obtain a **hyperplane**, a figure that we can no longer easily visualize.

Let’s now discuss how we might use assess and use the multiple linear regression model.

Using the Multiple Linear Regression Model

After the model has been fit, the next logical question is whether the model is useful or not. One way of assessing this is to determine if there is an overall regression relationship between $Y$ and the $x$ variables. We can examine the overall regression relationship by examining $R^{2}$ and by conducting the model F test. Let’s discuss the overall test first.

Model F Test

  1. Specify the hypothesis. The hypothesis is very similar to the ANOVA hypothesis of Chapter 14.

$$\begin{aligned} H_{0} &:&\beta_{1}=\beta_{2}=\ldots =\beta_{p}=0 \ H_{1} &:&\text{ Not all }\beta ^{\prime }s\text{ equal 0 [or, at least one }% \beta \text{ is not equal to }0]\end{aligned}$$

  1. Calculate the test statistic.

    The test uses an $F$ statistic

    $$f=\frac{SSR/p}{SSE/(n-(p+1))}$$

  2. Compute the $p$-value

    $$p=P(F_{p,n-(p+1)}>f)$$

    Alternatively, we can define the rejection region as follows:

    $${\text{ }f_{p,n-(p+1)}:f_{p,n-(p+1)}>f_{p,n-(p+1)},_{\alpha }}$$

  3. Make a conclusion

    If the $p$-value $\leq \alpha $, we reject $H_{0}.$ If the $p$-value $% >\alpha $, we fail to reject $H_{0}.$ Equivalently, if $f$ lies within the rejection region, we reject $H_{0}$. If $f$ lies outside the rejection region, we fail to reject $H_{0}.$

In general, to construct confidence intervals or tests for regression coefficients, we need the standard error and the appropriate value of $t$ from the Student’s t distribution. The test for an individual regression coefficient $\beta_{j}$ is as follows.

Test and Confidence Interval for Individual $\beta_{j}$

  1. Specify the hypotheses.

    The parameter of interest is $\beta_{j}$, so the hypotheses are statements about its value.

    $$\begin{aligned} \text{(a) }H_{0} &:&\beta_{j}\leq c\text{ vs. }H_{0}:\beta_{j}>c \ \text{(b) }H_{0} &:&\beta_{j}\geq c\text{ vs. }H_{0}:\beta_{j}<c \ \text{(c) }H_{0} &:&\beta_{j}=c\text{ vs. }H_{0}:\beta_{j}\neq c\end{aligned}$$

    In practice, $c$ is usually $0$, but it could be anything. In regression programs such as Excel, the hypothesis tested by default is

    $$H_{0}:\beta_{j}=0\text{ vs. }H_{0}:\beta_{j}\neq 0$$

  2. Calculate the test statistic.

    The test statistic is a $t$ statistic:

    $$t_{0}=\frac{b_{j}-c}{s.e.(b_{j})}$$

  3. Compute the p-value

    $$\begin{aligned} \text{(a) }p &=&P(t_{n-(p+1)}>t_{0}) \ \text{(b) }p &=&P(t_{n-(p+1)}<t_{0}) \ \text{(c) }p &=&2\min [P(t_{n-(p+1)}<t_{0}),P(t_{n-(p+1)}>t_{0})]\end{aligned}$$

    Alternatively, we can define the rejection region as follows:

    $$\begin{aligned} \text{(a)}{\text{ }t_{n-(p+1)} &:&t_{n-(p+1)}>t_{n-(p+1),\alpha }} \ \text{(b)}{\text{ }t_{n-(p+1)} &:&t_{n-(p+1)}<-t_{n-(p+1),\alpha }} \ \text{(c)}{t_{n-(p+1)} &:&|t_{n-(p+1)}|>t_{n-(p+1),\alpha /2}}\end{aligned}$$

  4. Make a conclusion

    If the $p$-value $\leq \alpha $, we reject $H_{0}.$ If the $p$-value $% >\alpha $, we fail to reject $H_{0}.$ Equivalently, if $t_{0}$ lies within the rejection region, we reject $H_{0}$. If $t_{0}$ lies outside the rejection region, we fail to reject $H_{0}.$

An important issue arises in multiple regression with respect to $R^{2}$ that we summarize in the following theorem.

The confidence interval for an individual coefficient is given in the following definition.

A $(1-\alpha )100%$ confidence interval for $\beta_{j}$ is given by

$$b_{j}\pm t_{n-(p+1),\alpha /2}s.e.(b_{j})$$

where $t_{n-(p+1),\alpha /2}$ is the value of Student’s $t$ distribution with $n-(p+1)$ degrees of freedom and $\alpha /2$ probability to the right, and $s.e.(b_{j})$ is the standard error of $b_{j}.$

The interpretation of a confidence interval in the multiple regression context must mention the other variables being held constant.

An important issue arises in multiple regression with respect to $R^{2}$ that we summarize in the following theorem.

$R^{2}$ will never decrease with the addition of $x$ variables. In other words, it is possible to “force” $R^{2}$ to $1$ by adding additional $x$ variables, whether they are useful for predicting $Y$ or not.

Therefore, what is needed is a measure of fit that penalizes us for adding additional variables that don’t really help in predicting $Y.$ Statisticians have defined many such measures, but the one most used in practice (and which is reported in Excel) is called adjusted $R^{2}.$

Adjusted $R^{2}$ (denoted as $R_{adj}^{2})$ is a penalizing measure of fit for a multiple regression model defined by

$$R_{adj}^{2}=1-\left( \frac{n-1}{n-(p+1)}\right) (1-R^{2})$$

Some facts about $R_{adj}^{2}$ that make it useful for assessing model fit are: 1). it is possible for $R_{adj}^{2}$ to decrease if the addition of an independent variable is not useful for predicting $Y$, and 2). it is possible for $R_{adj}^{2}$ to be negative. When fitting a multiple regression model, $R_{adj}^{2}$ is the preferred measure of fit over $R^{2}.$
